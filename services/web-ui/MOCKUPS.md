# PantryPal Web UI - Design Documentation

**Version:** 2.0.0
**Framework:** React + Vite
**Primary Color:** Orange (#d97706 light, #f59e0b dark)
**Design System:** Minimal, warm, functional

---

## Design System

### Color Palette

#### Light Theme
```
Primary: #d97706 (Warm orange)
Primary Dark: #b45309
Success: #10b981 (Green)
Danger: #ef4444 (Red)
Warning: #d97706 (Orange)
Info: #3b82f6 (Blue)

Text Primary: #292524 (Near black)
Text Secondary: #78716c (Gray)
Text Tertiary: #a8a29e (Light gray)

Background: #fef6ec (Warm beige)
Card: #fffcf7 (Off-white)
Card Hover: #fef6ec
Accent Background: #fef3c7 (Light yellow)

Border: #f5e6d3 (Light tan)
Border Dark: #e8d5bb

Expired Background: #fef2f2 (Light red)
Expired Text: #dc2626
Warning Background: #fffbeb (Light yellow)
Warning Text: #d97706
Good Text: #10b981
```

#### Dark Theme
```
Primary: #f59e0b (Brighter orange)
Primary Dark: #d97706
Success: #34d399 (Lighter green)
Danger: #f87171 (Lighter red)

Text Primary: #fafaf9 (Near white)
Text Secondary: #d6d3d1 (Light gray)

Background: #0c0a09 (Near black)
Card: #1c1917 (Dark gray)
Card Hover: #292524
Accent Background: #451a03 (Dark amber)

Border: #292524
Expired Background: #450a0a (Dark red)
Warning Background: #451a03 (Dark amber)
```

### Typography
- **Large Heading:** 32px, Bold
- **Heading:** 24px, Bold
- **Subheading:** 20px, Semi-bold
- **Body:** 16px, Regular
- **Small:** 14px, Regular
- **Caption:** 12px, Regular

### Spacing Scale
- **XS:** 4px
- **SM:** 8px
- **MD:** 12px
- **LG:** 16px
- **XL:** 24px
- **XXL:** 32px
- **XXXL:** 48px

### Border Radius
- **SM:** 6px
- **MD:** 8px
- **LG:** 10px
- **XL:** 12px
- **Full:** 9999px (pill shape)

### Shadows
**Light Theme:**
- Small: `0 1px 3px rgba(0, 0, 0, 0.05)`
- Medium: `0 2px 8px rgba(0, 0, 0, 0.08)`
- Large: `0 4px 12px rgba(217, 119, 6, 0.3)`

**Dark Theme:**
- Small: `0 1px 2px rgba(0, 0, 0, 0.5)`
- Medium: `0 2px 8px rgba(0, 0, 0, 0.6)`
- Large: `0 4px 16px rgba(0, 0, 0, 0.7)`
